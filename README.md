## Descripción del proyecto
Este repositorio contiene el código del dashboard y la documentación del proyecto.

## Instalación de librerías
1. Escribir el siguiente comando en la terminal/Powershell
```bash
pip install -r requirements.txt
```
Esto instalará las librerías necesarias para la correcta ejecución del proyecto

## Iniciar la aplicación
1. **Clonar repositorio**
```bash
git clone https://github.com/CodeAnd-Co/Documentacion.git
```
Este comando creará una carpeta llamada Ctrl-4_Reto_LogiMayab en tu directorio actual.

2. **Navegar al directorio de la aplicación**
```bash
cd app
```
Ahora te encuentras en el directorio donde reside la aplicación 

3. Iniciar la aplicación
- Si tu comando es python
```bash
python app.py
```
- Si tu comando es python3
```bash
python3 app.py
```
- Si tu comando es py
```bash
py app.py
```
Este comando iniciará la aplicación de Dash en el puerto ```http://localhost:8050/```
